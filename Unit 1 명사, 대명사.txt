[ 명사 생김새 ]
-tion, -sion, -ness, -sy, -ment, -ty, -sm, -ance(-ence)

ex) 명사를 찾아라.
1.
(A) appreciate (B) appreciative (C) appreciation (D) appreciates 

2.

3.
(A) enthusiasm (B) enthusiastically (C) enthusiastic (D) enthused

4. 
(A) participation (B) participant (C) participate (D) participatory

답)
1. C , 2. D , 3. A , 4. A(사물명사), B(사람 명사)


[ 명사 1초 공식 TOP 5 ]
1. 관사(a/an/the) + 명사 + 전치사
the ( purpose / purposeful ) of the event -> purpose
the ( project / projected ) before the deadline -> project

2. 전치사 + 명사 + 전치사
in ( comply / compliance ) with ~에 따라, ~에 응하여 -> compliance
in ( recognition / recognize ) of ~을 인정하여 -> recognition

3. 관사/소유격 + (부사) + (형용사) + 명사
a newly appointed (manager / manage) -> manager
my carefully planned(presently / presentation) -> presentation

4. 형용사(지시형용사, 수량형용사) + 명사
extra discounts / this bat / many (product / products) -> products

5. 명사 + 명사
photo (identify / identification) -> identification

[ 명사 자리는 주/목/보 ]
1) 주어 : The (information / informative) was not available. -> was가 동사이기 때문에 명사인 information
2) 타/목 : We encourage creative (innovation / innovate / innovative). -> innovation
3) 전/목 : Teamwork is crucial for successful (participate / participation). -> participation
4) 보어 : Customer satisfaction is a (reflect / reflected / reflecting / reflection) of our products quality. -> reflection

[ 가산명사 vs. 불가산명사 ]

가산명사는 셀 수 있는 명사
불가산명사는 셀 수 없는 명사

[표]
관사와 명사    |       가산명사       |          불가산명사
              | 단수명사  |  복수명사 |
a/an          |    O     |     X     |           X
the           |    O     |     O     |           O

[ 헷갈리는 가산명사 vs. 불가산 명사 모음 ]

가산 명사
task : 일

불가산명사

-ing 대부분 불가산 명사가 많다.

[ 사람명사 vs 사물명사 ]

사람 명사는 생긴 모양을 기억하면 쉽다. (-ee, -er, -or, -ist, -ent, -ant ...)
(표)
               |    사람 명사                       |  사물 명사
가산 vs 불가산  |   가산 명사                        |  가산/불가산 둘 다
단수           | a/an + 사람 명사                   |  - 
복수           | 사람명사 + -s                      |  -
예문           | We hired a qualified accountant.  |  She is intereted in accountanting.

사람은 셀 수 있다.(1명, 2명, 3명..)
셀 수 있기 때문에 사람 명사하면 가산명사다.
사람이 하나면 a/an 둘 이상이면 뒤에 -s
