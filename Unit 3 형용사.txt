[ 형용사 위치와 공식 ]  

1. 관/소 + 형용사 + 명사  
  a (succeed/successful) candidate  당선자  
  a는 관사 candidate는 명사니까 successful 형용사  
  her (previous/previously) health record  이전 건강 기록  
  her 소유격이고 health record는 복합명사니까 previous 형용사  

2. 주격 보어 자리(주어 + 2형식 동사 + 주격보어)  
  Health insurance has become (expense/expensive).  
  become은 2형식 동사니까 주격보어 expensive  
  The user manual was very (specify/specific).  
  -fy는 동사임. was는 2형식 동사니까 specific  

2형식 동사 : be, become, remain, seem, stay  

3. 목적격 보어 자리(주어 + 5형식 동사 + 목적어 + 목적격 보어)  
  The HR Department found the workshop (inform/informative).  
  found목적어 사용하고 informative목적격 보어 사용 inform은 동사니까 형용사인 informative 사용  
  The news made Mr. Tyler (happiness/happy).  
  5형식 동사 made이고 happy를 형용사 사용  

5형식 동사 : keep, find, consider, make  

[ 수량형용사 수일치 ]  

1. each, every, another + 단수명사  
I know every (student / students).  
모든 사람명사는 셀 수 있는 가산명사지만, 수량형용사 들어갔을 땐 관사를 신경쓰지 않는다.  
그러므로 단수명사 student 사용  
We need another (computer / computers) for the presentation.  
단수명사 computer  

예외)  
* every + 숫자 + 복수명사 : ~ 마다  
* another + 숫자 + 복수명사 : 또 다른 ~  
중간에 숫자 같은게 들어가면 뒤에는 복수명사 쓸 수 있다.  
ex) every 4 years : 4년마다  
ex) another 3 weeks : 또 다른 3주  

2. many, (a) few, a number of, several + 복수명사(명사 뒤에 s 붙인 것)  
She participated in a few (workshop / workshops) about communication skills.  
a few 뒤에 workshops  

3. much, little, a little, less + 불가산명사(셀 수 없는 명사)  
I had so much (trouble / troubles) because of the delayed schedule.  
much 뒤에는 trouble을 사용  

4. all, some, most, a lot of + 복수명사 / 불가산명사  
All (applicant / applicants) are required to answer questions.  
All 뒤에 applicant는 단수기 때문에 applicants  
(Every / All) information should be available.  
information은 불가산명사이다. 하지만 Every뒤에는 단수가 들어가야하므로 All  

a few / few | a little / little 차이  
a few / few + 셀 수 있는 복수명사  
a little / little + 셀 수 없는 불가산명사  
그리고 앞에 a가 붙으면 긍정적으로 해석, 없으면 부정적으로 해석  
ex) 조금 있는 가구 : a little furniture  
ex) 거의 없는 책 : few books  

[ 빈출 형용사 표현 암기 list ]  

considerable : 상당한       considerate : 사려 깊은  
expensive : 비싼            expansive : 광활한, 포괄적인  
favorable : 우호적인        favorite : 가장 좋아하는  
responsive : 반응하는       responsible : 책임 있는  
successful : 성공적인       successive : 연속의  
comparable : ~ 와 비슷한    comparative : 비교의  
reliable : 믿을만한         reliant : 의존적인  

이 문제는 해석으로 풀기
The recently appointed manager has (considerable / considerate) work experience.  
상당한으로 해석해야하니까 considerable  
In order to get (reliant / reliable) data, we need the follow the rules strictly.  
의존적인으로 해석해야하니까 reliable

주로 : mainly, Iargely
단지 : simply, only, just
이전에, 예전에 : formerly
원래 : orginally
한 때 : once

형용사는 관사가 앞에 있을 떄 관.형.명으로 명사를 수식해준다. 
하지만 관.명 이렇게 있으면 2글자 이상이며 주어, 동사가 없으니까 명사의 역할을 하는 구 즉, 명사구가 된다.

Originally a warehouse for agricultural produce, this building has been renovated and is now used as shared housing.
warehouse는 명사고 a는 관사. now가 들어갔을 때 originally가 자주 나옴. 이랬는데 지금은 이렇게 됐다.
The event spacem which is mainly a gallery for local artists, occasionally hosts community gatherings.
이 문장에서는 a gallery가 명사구이다.
Even a brief interruption in the power supply can disrupt the manufacturing process.
이 문장에서는 a brief interruption이 명사구
