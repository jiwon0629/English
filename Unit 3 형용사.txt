[ 형용사 위치와 공식 ]  

1. 관/소 + 형용사 + 명사  
  a (succeed/successful) candidate  당선자  
  a는 관사 candidate는 명사니까 successful 형용사  
  her (previous/previously) health record  이전 건강 기록  
  her 소유격이고 health record는 복합명사니까 previous 형용사  

2. 주격 보어 자리(주어 + 2형식 동사 + 주격보어)  
  Health insurance has become (expense/expensive).  
  become은 2형식 동사니까 주격보어 expensive  
  The user manual was very (specify/specific).  
  -fy는 동사임. was는 2형식 동사니까 specific  

2형식 동사 : be, become, remain, seem, stay  

3. 목적격 보어 자리(주어 + 5형식 동사 + 목적어 + 목적격 보어)  
  The HR Department found the workshop (inform/informative).  
  found목적어 사용하고 informative목적격 보어 사용 inform은 동사니까 형용사인 informative 사용
  The news made Mr. Tyler (happiness/happy).  
  5형식 동사 made이고 happy를 형용사 사용

5형식 동사 : keep, find, consider, make  

[ 수량형용사 수일치 ]  

* every + 숫자 + 복수명사 : ~ 마다  
* another + 숫자 + 복수명사 : 또 다른 ~  

1. each, every, another + 단수명사  
I know every (student / students).  
모든 사람명사는 셀 수 있는 가산명사지만, 수량형용사 들어갔을 땐 관사를 신경쓰지 않는다.  
그러므로 단수명사 student 사용  
We need another (computer / computers) for the presentation.  
단수명사 computer  

2. many, (a) few, a number of, several + 복수명사(명사 뒤에 s 붙인 것)  
She participated in a few (workshop / workshops) about communication skills.  
a few 뒤에 workshops

3. much, little, a little, less + 불가산명사(셀 수 없는 명사)  
I had so much (trouble / troubles) because of the delayed schedule.  
much 뒤에는 trouble을 사용

4. all, some, most, a lot of + 복수명사 / 불가산명사  
All (applicant / applicants) are required to answer questions.  
All 뒤에 applicant는 단수기 때문에 applicants
(Every / All) information should be available.
